HTML Table Reference

Table Tags

Tag Name Description
<table> Table Represents a two-dimensional table made of rows and columns.
<thead> Table Head Defines the headings of a table columns encapsulated in table rows.
<tbody> Table Body Contains all table data other than 'table heading' and 'table footer' content.
<tr> Table Row Adds rows to a table before adding 'table data' <td> and 'table headings' <th>.
<td> Table Data This element can be nested inside a table row element <tr>, to add a cell of data to a table <td>.
<tfoot> Table Foot This element uses table rows to give footer content or to summarize content at the end of a table.

Table Attributes

Attribute Name Description
colspan Column Span The colspan attribute on a table header <th>, or tabledata <td> element
indicates how many columns that particular cell should span within the table.
rowspan Row Span Similar to colspan, the rowspan attribute on a tableheader or table data element
indicates how many rowsthat particular cell should span within the table.
